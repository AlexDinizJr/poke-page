# 🧩 PokePage

PokePage é uma aplicação fullstack para catalogação de Pokémons, com foco em consumo de API externa, persistência de dados e construção de uma aplicação web completa.

---

## 🎯 Objetivos

- Consumir dados de uma API pública de Pokémons  
- Persistir os dados localmente (MongoDB)  
- Reduzir dependência da API externa  
- Disponibilizar um catálogo consultável  

---

## 🏗️ Arquitetura

A aplicação é composta por:

- **Frontend (React)**
- **Backend (ASP.NET Web API)**
- **Banco de Dados (MongoDB)**

---

## 🛠️ Stack

- React + TypeScript  
- ASP.NET Web API  
- MongoDB
